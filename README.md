# Web Scraping Portfolio

Python-парсеры для сбора данных с сайтов. Выгрузка в Excel и CSV.

## Скрипты

| Файл | Сайт | Что собирает |
|------|------|--------------|
| rusprofile_test.py | rusprofile.ru | ИНН, ОГРН, адрес, руководитель организации |
| book parse.py | books.toscrape.com | Названия и цены книг (пагинация) |
| hockey parse.py | scrapethissite.com | Статистика хоккейных команд |
| selenium_tovari.py | webscraper.io | Товары, цены, рейтинги (Selenium) |
| wikicntry parse.py | wikipedia.org | Страны, население, регион |

## Стек
requests, BeautifulSoup, Selenium, pandas
